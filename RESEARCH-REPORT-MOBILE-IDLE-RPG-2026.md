# 모바일 MMORPG·방치형 RPG 트렌드 조사 보고서

> 시즌2 적용 결정(2026-07-18): 조사 사례의 계정 레벨 구조는 채택하지 않는다. 현재 게임은 카드 등급·강화·종족 시너지·도감 보너스만 전투력에 반영한다. 이 문서의 계정 레벨 언급은 시장 사례 설명일 뿐 구현 기준이 아니다.

> 조사 기준일: 2026-07-16
> 목적: 기존 카드뽑기로 획득한 카드를 스트리머와 팬이 함께 활용하는 게임 기획
> 제외 범위: 과금, 광고, 상품, 수익화 설계
> 상태: 기획 검토용. 구현·커밋·push·배포 대상 아님

## 1. 조사 방식

- 모바일 MMORPG 플레이 구조 조사 모델
- 모바일 방치형·수집형 RPG 조사 모델
- 카드 컬렉션·중복·커뮤니티 설계 조사 모델
- 메인 분석에서 세 결과를 현재 카드 데이터와 웹게임 규모에 맞춰 교차 검토
- 현재 기준 카드: 총 212장, 인물 22명, `F~SSS` 204장, 전투 불가 `EX` 8장

## 2. 최종 결론

우리 게임은 모바일 MMORPG를 만들면 안 된다. 대신 MMORPG의 `서버 공동체 재미`만 가져온 방치형 카드 RPG가 적합하다.

권장 정의:

> 평일에는 팬이 보유 카드 5장으로 짧게 성장하고, 주말·방송일에는 모든 팬의 원정대가 하나의 서버 보스를 함께 쓰러뜨리는 비동기 수집형 방치 RPG.

핵심은 다음 네 가지다.

1. 카드뽑기 결과가 즉시 게임 전력과 편성 선택으로 연결된다.
2. 212장을 각각 키우지 않고 `계정 공용 성장`으로 카드 교체 부담을 없앤다.
3. 도감은 영구 성장 동기를 주되 전투력 격차를 고착시키지 않게 상한을 둔다.
4. 실시간 동시 전투 대신 서버 공동 HP, 개인 기여, 스트리머 지휘와 방송 연출로 함께 싸우는 느낌을 만든다.

## 3. 2023~2026 핵심 트렌드

### 모바일 MMORPG

- 자동 사냥과 무접속 성장은 유지되지만 중요한 순간만 직접 참여하는 구조가 강해졌다.
- 상시 오픈월드 난전보다 짧은 길드 리그, 시즌 월드전, 인스턴스 보스가 늘었다.
- 신규·복귀 유저를 별도 성장 부스트, 전투력 제한, 시즌 규칙으로 빠르게 합류시킨다.
- 대규모 전투 자체보다 길드 정체성, 공동 목표, 보스 페이즈, 역전 상황과 순위 변화가 방송 볼거리를 만든다.
- 과도한 레이드와 반복 숙제는 삭제·스킵·즉시완료 대상으로 바뀌는 추세다. 아이온2는 2026년 시즌 업데이트에서 피로 감소를 이유로 일부 레이드를 제거하고 성장 부스트와 즉시완료를 확대했다. [NC 공식 발표, 2026-04-08](https://about.ncsoft.com/en/news/article/aion2_update_260408)

### 방치형·수집형 RPG

- 최고 스테이지가 오프라인 보상 효율을 결정한다.
- 개별 캐릭터 레벨보다 계정 공용 레벨 또는 성장 슬롯을 사용한다.
- 자동 편성만 강요하지 않고 `카드 일부 고정 + 나머지 자동 완성`을 지원한다.
- 중복 카드는 강화 후에도 초과분을 범용 조각이나 다른 성장에 재활용한다.
- 2025~2026 운영 방향은 일일 콘텐츠 추가보다 스킵, 프리셋, 주간 통합, 백그라운드 진행에 가깝다.
- 동시 접속 협동보다 각자 원하는 시간에 공격하고 결과가 서버 전체에 누적되는 비동기 공동전이 적합하다.
- Seven Knights Idle Adventure는 오프라인 보상, 다인 편성, 캐릭터 특성을 이용한 덱 조합을 장르 핵심으로 제시한다. [Netmarble 공식 게임 소개](https://skidle.netmarble.com/en/game)
- MapleStory: Idle RPG는 2026년 보스·길드 레이드, 성장 구간과 난도를 선택하는 콘텐츠, 빠른 추격 이벤트를 함께 확장했다. [Nexon 개발자 노트, 2026-04-16](https://forum.nexon.com/maplestoryidle/board_view?board=6653&thread=3424530)
- AFK Journey는 긴 공백을 줄이기 위해 시즌을 여러 단계로 나누고, 서버 최초 진행이 전체 서버 보상으로 이어지는 구조를 도입했다. [Farlight 개발자 발표, 2025-03-17](https://afkjourney-news.farlightgames.com/pt_BR/detail/83)

## 5. 권장 게임 구조

### 기본 루프

1. 카드뽑기로 새 카드와 중복 카드를 얻는다.
2. 보유 카드 5장으로 원정대를 편성한다.
3. 원정대가 스테이지를 자동 진행한다.
4. 최고 스테이지에 따라 오프라인 보상을 받는다.
5. 계정 성장과 카드 강화를 진행한다.
6. 도감 세트를 완성해 작은 영구 보너스를 얻는다.
7. 주간 서버 보스에 원정대를 출전시킨다.
8. 방송에서 서버 진행도, 팬 카드, 하이라이트와 최종 결과를 함께 본다.

### 접속 리듬

- 평일 2~4분: 방치 보상 일괄 수령, 성장 1회, 막힌 스테이지 재도전
- 주 2~3회 10분: 덱 조정, 시련 던전, 서버 보스 도전
- 방송일 20~30분: 스트리머 투표·지휘, 보스 페이즈 전환, 결과 발표
- 오프라인 보상 상한: 24시간 권장
- 이미 클리어한 전투: 즉시 스킵
- 일일 숙제: 여러 메뉴가 아니라 주간 활동 게이지 하나

## 6. 전투와 편성

### 권장안

- 5장 편성, `2×3` 배치판
- 카드 개별 레벨 없음
- 계정 레벨 또는 5개 성장 슬롯을 모든 카드가 공유
- 카드 역할은 인물 또는 카드 태그 기반 4종만 사용
  - 수호
  - 근접
  - 원거리
  - 지원
- 카드별 차이는 등급 기본 능력, 강화 수치, 패시브 1개로 제한
- 자동전투지만 배치 위치, 종족 조합, 보스 약점으로 편성 판단을 남긴다.

### 편성 방식

1. 즉시 자동 편성
2. 원하는 카드 1~4장 잠금 후 나머지 자동 완성
3. 완전 수동 편성

자동 편성은 단순 전투력순이 아니라 `전열 부족`, `보스 약점`, `종족 보너스` 같은 추천 이유를 표시해야 한다. MARVEL SNAP도 사용자가 일부 카드를 고정한 뒤 나머지를 자동 완성하는 방식을 지원한다. [공식 패치, 2023-12-05](https://marvelsnap.com/patch-notes-dec-05-2023/)

## 7. 도감 컬렉션 권장안

도감 영구 스탯은 유지한다. 단, 카드 한 장마다 퍼센트를 무한 합산하면 안 된다.

### 구조

- 최초 획득 시 도감 영구 등록
- 분해·강화 재료 사용 후에도 기록 유지
- 인물, 종족, 등급, 전체 수집률 세트 구성
- 세트는 `3/6/9장`처럼 중간 단계 보상 제공
- 완전 수집 보상은 큰 전투력이 아니라 프레임, 칭호, 연출 중심
- 출시 시 도감 전체 전투 보너스 상한 `5%` 권장
- 서버 보스·랭킹에서는 도감 보너스 효과를 추가 축소
- 신규 카드가 추가될 때마다 최대 전투력이 자동 증가하지 않게 구간을 수동 확장

가디언 테일즈는 컬렉션을 작은 계정 영구 스탯으로 사용하면서 기간 한정 협업 대상을 컬렉션 전투 보너스에서 제외한다. [공식 패치, 2026-02-17](https://guardiantales.com/news/4751), [협업 안내, 2024-02-07](https://guardiantales.com/news/4168?lang=en)

### EX 처리

- 전투, 편성, 강화, 도감 전투 스탯에서 완전 제외
- 프로필 전시대 3칸
- 획득일, 획득 이벤트, 서버 기록 배지
- EX 3/6/8장으로 전시 배경과 연출 해금
- 서버 보스 막타, 시즌 기록, 방송 이벤트와 연결

EX는 강함이 아니라 `그때 그 이벤트에 참여했다`는 기록이어야 한다.

## 8. 낮은 등급과 중복 카드

### 낮은 등급

낮은 등급을 강제로 한 장 넣게 하면 숙제가 된다. 다음 방식이 낫다.

- 등급별 편성 코스트 모드
- 저등급의 짧은 재사용 시간 또는 보조 패시브
- 매주 바뀌는 `F~C 카드 효과 증가`, `저그 코스트 감소` 같은 규칙
- 특정 카드 한 장을 필수로 지정하지 않음

명일방주의 일부 모드는 낮은 등급 유닛의 편성 비용을 낮춰 실제 출전 가치를 만든다. 낮은 등급을 단순 재료가 아니라 편성 경제성으로 살리는 사례다. [공식 업데이트, 2025-08-07](https://www.arknights.global/news/1501)

### 중복 카드

- 현재 확정된 `+9 강화`의 재료로 사용
- 마지막 보유 1장은 자동 보호
- 잠금 카드는 재료에서 자동 제외
- `+9` 이후 초과 중복은 등급별 기억 조각으로 변환
- 기억 조각은 같은 등급 이하 카드에만 사용
- 초과 중복을 무제한 범용 재화로 바꿔 상위 카드만 몰아 키우는 구조는 금지

기존 강화 확률과 파괴 규칙은 별도 기획으로 유지하되, 방치 RPG의 기본 진행이 `+7 이상 강화`를 요구하게 만들면 안 된다.

## 9. 서버 공동 콘텐츠

### 주간 서버 보스

- 주말 72시간 또는 주간 상시 개방
- 모든 팬이 원하는 시간에 최대 3회 도전
- 공격 결과를 서버 공동 HP에 누적
- 브라우저를 계속 켜둘 필요 없음
- 개인 최고 피해와 전체 누적 기여를 분리
- 1회만 참여해도 공동 목표 참가자로 인정
- 신규 유저는 현재 성장 구간 대비 성과로 별도 표시
- 랭킹 보상은 전투력보다 칭호, 프레임, 전광판 노출 중심

블루 아카이브식 비동기 보스 구조는 각자 공격하면서도 전체 이벤트를 함께 진행하는 감각을 만든다. [Nexon 공식 Total Assault 가이드, 2025-11-26 갱신](https://forum.nexon.com/bluearchive-en/board_view?thread=2523158)

### 방송 연동

- 스트리머가 보스 변이 3개 중 하나를 선택하거나 투표 진행
- 선택지는 난도와 보상 차이가 아니라 공략 방식만 변경
- 스트리머 대표 덱을 팬이 하루 1회 지원 카드로 대여
- 페이즈 전환 때 최고 피해, 막타, 특이 편성 자동 하이라이트
- 참여자 대표 카드가 보스 화면 배경에 행렬로 등장
- 실시간 채팅 파싱은 후순위

여기서 스트리머는 운영자가 아니라 `서버 이벤트 지휘자` 역할이다.

- 카드마다 완전히 다른 복잡한 액티브 스킬
- 일일 던전 10종과 별도 입장권
- 장비, 룬, 유물, 펫, 연구를 동시에 여는 다중 성장
- 전체 서버 절대 순위만 있는 경쟁
- 랭킹 상위 보상이 다음 랭킹 전투력을 높이는 구조
- EX와 기간 한정 카드가 있어야 완성되는 전투 도감
- 특정 시간에 결석하면 진영 전체가 손해 보는 규칙

## 12. 기획안 2 권장 확정 방향

| 항목 | 권장 결정 |
|---|---|
| 장르 | 서버 공동 콘텐츠가 있는 수집형 방치 카드 RPG |
| 전투팀 | 5장 |
| 성장 | 계정 공용 성장 + 카드 +9 강화 |
| 전투 | 자동, 배치·태그·보스 약점만 전략화 |
| 오프라인 | 최대 24시간 |
| 도감 | 영구 스탯, 전체 상한 5%, 완성은 명예 보상 |
| EX | 전투 불가, 전시·기록 전용 |
| 핵심 공동전 | 주간 비동기 서버 보스 |
| 방송 역할 | 보스 규칙 선택, 대표 덱 지원, 결과 중계 |
| 시즌 | MVP 이후 6~8주 단위로 검토 |
| 실시간 멀티 | MVP 제외 |

## 13. 판단

기획안 2는 기존 기획안 1보다 현재 카드가챠의 후속작으로 구현 가능성이 높다.

- 기존 카드와 계정 데이터를 가장 많이 재사용한다.
- 팬이 혼자 접속해도 진행 가능하다.
- 방송 중에는 서버 전체가 한 장면에 모인다.
- 실시간 매칭 인원이 부족해도 콘텐츠가 멈추지 않는다.
- 운영자가 매일 새 맵과 몬스터를 제작하지 않아도 보스 규칙과 시즌 변형으로 갱신할 수 있다.

가장 중요한 차별점은 방치 전투가 아니다. `내가 뽑은 스트리머 카드가 팬 전체의 서버 보스전에 실제로 등장하고 기록으로 남는 것`이다.
