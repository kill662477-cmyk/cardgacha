# PDB-2 전투·성장·경제 밸런스 잠금

> 상태: 통과
> 밸런스 버전: `2026.07.18-random-loot-1`
> 단일 원본: `src/renewal/config.js`

## 핵심 변경

- 계정 레벨과 계정 EXP를 게임 상태, 보상, 전투 공식과 UI에서 제거했다.
- 삭제 전 `1.03^(계정 Lv-37)` 배율은 30일 시뮬레이션에서 약 3만 배까지 커져 카드 성장보다 영향이 컸다.
- 전투 성장은 카드 등급, 강화, 종족 시너지와 도감 보너스로만 발생한다.
- 표시 전투력은 공격속도, 치명타, 일반·보스 패시브, 회복과 약화를 포함한다.
- 광역 보너스는 일반 웨이브에만, 보스 보너스는 보스에만 적용한다.
- 각 전투 등급에는 8개 유형을 모두 넣고 유형별 카드 수 차이를 최대 1장으로 제한한다.
- 구 `schemaVersion: 1` 저장값의 `accountLevel`, `accountExp`는 v2 변환에서 삭제한다.

## 전투 성장

- 강화 능력치 배율: `1, 1.12, 1.27, 1.44, 1.63, 1.85, 2.1, 2.38, 2.7, 3.0`.
- 종족 시너지: 동일 종족 3~4장 공격·체력 5%, 5장 공격·체력 12%.
- 종족 시너지 공격 배율은 공격에만 적용하며 방어력에는 곱하지 않는다.
- 증폭형은 카드당 편성 전체 공격 4%를 올린다.
- 카드 ID 개별 편차는 ±3%. 같은 유형·같은 성급에서는 상위 등급 최저값이 하위 등급 최고값보다 높다.
- 도감 전투 보너스 합계 상한 100%. 현재 전투 카드 204장 전체 등록 결과는 78.75%.
- 도감 방치 보너스 최대 30%.
- S 9성 또는 SS 5성 5장은 도감·계정 배율 없이 50단계를 완주한다.
- D·E·F 혼합 9성 편성은 최종 지역에서 막힌다.
- 0성 동일 등급 기준 도달 단계: F 3, E 5, D 8, C 10, B 11, A 18, S 24, SS 33, SSS 40.
- 고강화 저등급 카드가 저강화 고등급 카드를 역전하는 것은 의도된 성장 보상이며 허용한다.

## 카드 등급 배치

- 총 212장: 전투 카드 204장, 전시 전용 EX 8장.
- SSS 14장은 사용자 지정 ID를 고정하며 자동 재배치하지 않는다.
- F·E·D·C·B·A·S는 각 24장, SS는 22장이다.
- 삭제 사진 8장은 카드 데이터, 도감과 카드팩 추첨 풀에서 제거했다.
- 신규 사진 10장은 사진 구도와 완성도를 확인한 뒤 A~SS로 배치했다.
- 김윤환은 비SSS 5장 중 SS 4장·S 1장, 토마토와 지두두는 각각 SS 3장·S 3장이다.
- 김민철과 변현제는 각각 SS 2장·S 1장, 나머지 남성 코치는 최소 A 이상 1장을 유지한다.

## 모험·보상

- 모든 런은 `1-1`에서 시작하고 실패 또는 50단계 완료 시 종료한다.
- 첫 런 시작부터 4시간 동안 새 런 최대 3회.
- 모험 포인트: `45n + 5n(n+1)`P, 10단계 1,000P, 50단계 15,000P 상한.
- 모험 카드 EXP: 클리어 1단계당 편성 카드 각각 1.
- 오프라인 카드 EXP: 분당 `0.04 + 현재 단계×0.004`, 최대 24시간.
- 30분·2시간 EXP 버프가 실제 겹치면 버프 추가분 최소 1 EXP를 보장한다.
- 빠른 전투: 행동력 20, 하루 3회, 모험 런 횟수도 1회 차감.
- 행동력: 최대 120, 6분당 1 회복.

## 강화

- 1~3성 확정 성공.
- 4~9성 기본 성공률 `80, 70, 60, 50, 40, 30%`에서 등급 페널티 차감.
- 7~9성 파괴율 `3, 8, 15%`.
- 파괴 판정은 원본 카드를 삭제하지 않는다. 강화 단계와 카드 EXP만 0으로 초기화한다.
- 재료, 사용 아이템과 +9 시도 5,000P는 성공·실패·파괴 모두 소비한다.
- 카드 EXP가 현재 단계 최대치여야 강화할 수 있다.
- 강화 재료는 등급 규칙에 맞는 중복 카드이며 마지막 보유 1장은 보호한다.

## 경제

- 미니게임: 게임별 하루 최대 5,000P, 2종 합계 최대 10,000P.
- 카드 짝맞추기: 4×4 완료 500P, 6×6 완료 1,500P.
- 캄몬사과게임: 기본 40P + 제거 점수, 1회 최대 240P.
- SOOP: 별풍선 1개당 3P. 성장 시뮬레이션에서는 제외한다.
- 카드팩 확률과 가격은 이번 리밸런싱에서 변경하지 않았다.
- SSS 확률: 일반 0.006%, 정예 0.012%, 프리미엄 0.06%, 종족 0.0006%.

## 월드보스

- 매일 KST 17·18·19·20시, 회차당 30분 전투 + 30분 결과, 회차별 무료 도전 3회.
- 공동 HP `5,000,000,000`.
- 서버 기준 자동 피해 `2,766,667/초`: 30분 동안 약 49.8억. 개인 누적 2,000만이 성공 경계다.
- 로컬 샘플 편성 1회 피해 약 946만, 3회 누적 시 최대 개인 보상 도달.
- 성공 보상: 참여 1,000P, 200만 2,000P, 500만 3,500P, 1,000만 5,500P, 1,500만 8,000P, 2,000만 10,000P.

## 랜덤 추가 드롭

- 모험런·빠른 전투: 클리어 1/10/20/30/40/50단계 기준 드롭률 18/24/30/36/43/50%.
- 월드보스 보상 수령: 실패 35%, 성공 60%.
- 드롭 성공 후 카드팩 교환권 선택 비중은 모험 최고 구간 30%, 월드보스 실패 15%, 성공 25%.
- 카드팩 교환권 비중: 일반 55%, 정예 27%, 종족 15%, 프리미엄 3%.
- 나머지는 기존 상점 아이템 가중치로 지급. 한 판·한 회차당 최대 1개.
- 오프라인 보상에는 적용하지 않는다.
- 도전 1회마다 편성 카드 EXP 25.

## 성장 시뮬레이션

모든 프로필은 0성에서 시작한다. 카드 EXP, 등급별 중복 재료 기대량, 강화 성공률과 +9 포인트 비용을 반영한다. SOOP 후원과 뽑은 카드로 편성을 교체하는 효과는 제외한다. +7~+9에는 파괴 차단제를 사용한다고 가정한다.

랜덤 추가 드롭은 확률표와 기대 팩 획득률을 별도 검증하며, 소비 시점과 사용 가치가 계정마다 달라 30일 성장 시뮬레이션 수치에는 포함하지 않는다.

| 프로필 | 1일 | 7일 | 18일 | 30일 | 첫 완주 |
|---|---:|---:|---:|---:|---:|
| 하위 신규 | 2단계 / 78,092 | 7단계 / 99,185 | 11단계 / 127,306 | 16단계 / 144,484 | 미완주 |
| 중위 일반 | 12단계 / 130,162 | 33단계 / 206,880 | 42단계 / 302,069 | 50단계 / 380,766 | 28일 |
| 상위 집중 | 15단계 / 141,832 | 40단계 / 255,886 | 50단계 / 414,949 | 50단계 / 414,949 | 13일 |

- 자연 행동력 사용량은 하루 40·120·240으로 모두 자연 회복 240 이하.
- 하위는 30일에도 2지역에서 멈추고, 중위는 한 달 안에 완주, 상위 집중은 10~18일 목표 구간 안에서 완주한다.
- 기대값 모델이므로 라이브 강화 파괴, 실제 카드 교체와 이용 패턴은 PDB-1 플레이테스트에서 다시 확인한다.

## 기타 확정 규칙

- EX 8장은 카드팩·전투·강화·도감 스탯에서 제외한다.
- 최고기록 5·10·15·20·25·30·40·50 최초 달성 시 EX를 1장씩 지급한다.
- 시즌1 FUR 일반 인물 카드는 SSS, 김윤환 기존 배치는 유지, 단체사진은 EX 유지.
- 모든 카드팩과 작전 지원 보급팩 확률 합계는 100%다.

## 변경 통제

등급·전투 공식·모험·강화·재료·팩·행동력·도감·미니게임·SOOP·EX 규칙 변경 시:

1. `BALANCE_VERSION` 증가.
2. `npm.cmd run test:renewal` 실행.
3. `npm.cmd run simulate:renewal-balance` 실행.

사진 staging 재동기화는 `npm.cmd run sync:renewal-card-staging`으로 삭제·추가·F~SS 재배치·콘텐츠 생성을 한 번에 수행한다.

월드보스 회차 ID, 공동 HP, 회차 시간, 서버 자동 피해와 보상 구간은 회차 시작 전에만 운영 조정 가능하다.
