# 리뉴얼 UI·UX·브랜드 자산 감사

> 감사 기준: 2026-07-17 로컬 시즌2 화면, 현재 진입점 `index.html`
> 확인 화면: 모험, 상점, 강화, 도감, 월드보스, 미니게임, 랭킹
> 확인 해상도: `1366x768`, 모바일 가로 `844x390`

## 디자인 기준

- 게임명: `카드가챠 시즌2`
- 성격: 방송국 카드 IP 기반 다크 SF 방치형 RPG
- 중심: 실제 인물 카드가 가장 강한 시각 요소
- 톤: 검정·차콜 기반, 산성 라임은 명령과 진행 상태에만 사용
- 밀도: 게임 HUD 수준의 높은 정보 밀도. 주요 수치는 작지 않게 유지
- 금지: 임시 텍스트 로고, 빈 검은 이미지, 시즌 1 자산과 SF 자산의 무질서한 혼용

## 현재 판단

기능 화면 구조와 핵심 배경은 확보됐다. 최종 브랜드와 제품 완성도는 아직 아니다.

잘 된 부분:

- 월드보스와 강화실 배경은 화면 목적과 분위기가 명확하다.
- 카드가 실제 인물 중심으로 표시되고 등급 테두리 구분도 작동한다.
- PC 7개 화면의 정보 구조가 대체로 일관된다.
- 모바일 가로에서 주요 기능 접근과 카드 5장 표시는 유지된다.

주요 부족:

- 기존 카드백의 별·원형 몬스터·뿔·날개를 다크 SF 심볼로 재해석한 로고 시안이 적용됐다. 최종 시각 승인 전 상태다.
- 전체 UI가 검은 판넬과 7~9px 보조 글자에 과도하게 의존한다.
- 모험 전투에서 중앙 카드, 피해 숫자와 하단 카드가 겹쳐 전투 가독성이 낮다.
- 상점의 시즌 1 카드팩 이미지가 새 SF 상점·아이템 이미지와 시각적으로 충돌한다.
- 도감 미등록 카드가 빈 검은 박스로 보여 수집 욕구와 정보성이 약하다.
- 랭킹 TOP 3가 넓은 빈 상자 위주라 보상감과 경쟁 연출이 약하다.
- 미니게임 대기 화면이 공용 게임패드 아이콘과 `GAME READY`만 보여 개별 게임 개성이 부족하다.
- 오류, 빈 상태, 로딩, 첫 진입과 튜토리얼의 전용 시각 자산이 없다.

- 라임은 활성 상태와 주요 행동에 집중한다.

## P0: 카드 프레임 이미지 재제작

현재 리뉴얼 카드는 별도 프레임 이미지가 없고 CSS의 등급색 선과 내부 선으로만 표현된다. 실제 인물 카드가 게임의 핵심이므로 프레임을 필수 브랜드 자산으로 새로 제작한다.

제작 대상:

- 세로형 `F`, `E`, `D`, `C`, `B`, `A`, `S`, `SS`, `SSS` 9종
- 단체사진·박제용 EX 가로형 1종
- 미등록 카드용 공통 실루엣 프레임 1종
- 작은 썸네일에서도 등급을 구분할 수 있는 간소화 버전

고정 규격:

- 세로형 비율 `5:7`, 기준 원본 `2000x2800`
- EX 가로형 비율은 실제 단체사진 비율 검수 뒤 한 규격으로 고정
- 프레임은 투명 배경 오버레이 이미지로 제작
- 직선 프레임은 전체 너비의 3.4%, 모서리 결합부는 최대 4.2% 이내, 카드 경계 밖으로 돌출 금지
- 인물 얼굴·머리·손을 가리는 굵은 상단 장식 금지
- 하단 정보는 빈 여백을 따로 만들지 않고 사진 위 반투명 정보판으로 처리
- 등급, 이름, 종족과 강화 수치는 이미지에 굽지 않고 HTML 텍스트로 표시
- 모든 등급의 사진 안전영역과 이름표 위치를 동일하게 유지
- 이미지 프레임 바깥에 CSS 등급색 테두리나 내부 보조선을 중복해서 두르지 않는다. 선택 상태도 직사각형 선 대신 밝기와 발광으로 구분한다.
- 등급은 좌상단 검정 네온 코어 안에 표시하고, 강화는 우상단에 금속 별 아이콘과 `×N`만 띄운다.
- 강화 표시에 검정 배경, 테두리, 각진 배지 박스를 사용하지 않는다. 숫자 가독성은 얇은 외곽선과 단계별 발광으로 확보한다.
- 강화 사용자 표기는 `1성~9성`, 카드 위 축약 표기는 별 아이콘 `×1~×9`로 통일한다.

등급별 방향:

- `F~C`: 얇은 산업용 합금 프레임. 장식 최소
- `B~A`: 모서리 결합부와 약한 에너지 라인 추가
- `S`: 고유 발광 회로와 상단 등급 코어
- `SS`: 이중 에너지 라인과 깊이감 추가
- `SSS`: 가장 화려하지만 사진 면적은 다른 등급과 동일하게 유지
- `EX`: 전투 장비가 아닌 기록 보관소·명예 전당 성격의 가로형 프레임

구현 방식:

1. 공통 형상 마스터 1종을 먼저 제작하고 두께·안전영역 승인
2. 승인된 형상을 기준으로 F~SSS 재질·빛·장식만 단계적으로 확장
3. 모든 카드 화면이 동일한 `card-frame` 오버레이 컴포넌트를 사용
4. 모험, 강화, 도감, 월드보스, 랭킹, 미니게임과 카드팩 결과에 적용
5. 작은 카드에서는 간소화 이미지를 사용해 선이 뭉개지지 않게 처리

완료 기준:

- 카드 외곽 크기와 `5:7` 비율이 모든 화면에서 동일하다.
- 프레임이 카드 밖으로 넘거나 잘리지 않는다.
- 프레임 때문에 사진이 축소되거나 얼굴이 가려지지 않는다.
- 최저 등급은 절제되고 `S`, `SS`, `SSS`는 단계별 화려함이 분명하다.
- 모바일 72px 폭 카드에서도 등급과 이름이 식별된다.
- 기존 시즌1 SVG 카드 프레임은 제거했고 시즌2 WebP 프레임만 사용한다.

## P0: 화면별 UX 정리

### 모험

- 중앙 전투 카드 표시 크기 축소 또는 위치 재설계
- 피해 숫자 동시 노출량, 크기와 수명 제한
- 적 이름, 웨이브·보스 상태와 타깃 정보 강화
- 실패 배너가 카드와 피해 숫자를 가리지 않게 재배치
- 자동 진행, 빠른 전투와 보상 받기의 우선순위 재정리

### 상점

- 일반·정예·프리미엄·종족팩을 같은 세계관의 신규 패키지 자산으로 통일
- 카드팩과 지원 보급품의 색·실루엣 계열 분리
- 선택 상품, 가격, 1회·10회 구매와 확률 정보의 시선 순서 개선
- 구매 불가, 보유 교환권과 결과 요약 상태 추가

### 강화

- 현재 강한 배경과 챔버는 유지
- 대상 카드, 재료, 성공률과 위험 경고의 우선순위 강화
- 성공·실패·파괴 상태가 색뿐 아니라 형태·문구로도 구분되게 수정
- 강화 성공 연출 개선 완료: 4단계 3.12초, 이중 충격파, 회전 링, 방사 광선, 강화 구간별 입자·효과음 차등 적용
- 9성 `STAR UP` 표식은 카드 우상단 강화 배지와 겹치지 않도록 카드 바깥에 배치
- 데스크톱 1280×720과 모바일 가로 932×430에서 카드 식별, 안전 영역, 오버플로와 콘솔 오류 검증 완료

### 도감

- 미등록 카드 전용 실루엣·노이즈 자산 제작
- 보유·미보유 차이를 빈 검은 박스가 아닌 수집 상태로 표현
- 컬렉션 진행률과 실제 스탯 보너스의 연결 강화
- EX 전시 영역을 일반 카드 목록과 시각적으로 분리

### 월드보스

- 현재 보스와 전장 품질을 기준점으로 유지
- `ERROR` 공용 표식을 보스 고유 명칭·페이즈 표식으로 교체
- 참가·공격·보상 상태의 버튼 위계와 우측 작은 글자 개선
- 보스 피해, 서버 HP와 개인 기여의 시각적 차이 강화

### 미니게임

- 카드 짝맞추기와 캄몬사과게임의 개별 대표 이미지·아이콘 제작
- 대기, 진행, 성공, 시간 종료 화면을 게임별로 구분
- 보상 모드와 연습 모드 차이를 시작 전에 명확히 표시
- 모바일 드래그·뒤집기 영역과 남은 시간 가독성 강화

### 랭킹

- TOP 3 전용 금·은·동 프레임, 아바타 또는 대표 카드 추가
- 1위 중심의 크기·빛·모션 차등
- 내 순위, TOP 50 차이와 현재 편성을 한 덩어리로 읽히게 재구성
- 일반 순위표는 장식보다 스캔 속도를 우선
